Is it possible to migrate to Elastic Cloud from MS Hyper-V, XenServer hypervisors?

Elastic cloud is based on VMware virtualization Migration to VMware is possible not only from other virtualization systems (MS Hyper-V, XenServer), but also from "iron" servers without virtualization. This is a more complex and time-consuming process than migrating within a single virtualization system. But #CloudMTS has experience migrating client systems from physical servers without virtualization and Hyper-V infrastructure.
